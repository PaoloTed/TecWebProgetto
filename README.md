# Tecnologie Web - 2024/2025

Questa repository contiene tutto il materiale, gli esercizi e i progetti relativi al corso di **Tecnologie Web** del terzo anno d'università.

## 📂 Struttura della Repository

La repository è organizzata nelle seguenti cartelle principali:

- **`streetcats/`**: Contiene il progetto d'esame. È suddiviso in due componenti principali:
  - `streetcats-backend/`: Il codice lato server (API REST con Node.js, ecc.).
  - `streetcats-frontend/`: Il codice lato client e l'interfaccia utente.
- **`Esercizi/`**: Contiene gli esercizi e le prove pratiche svolti in preparazione e durante il corso (suddivisi per numero di assegnamento, ad es. `Es_1`, `Es_2`, ecc.).
- **`Materiale/`**: Materiale didattico, esempi di codice e approfondimenti forniti durante il corso.
- **`slide/`**: Le slide ufficiali delle lezioni.
- **`ProgettiStudenti/`**: Contiene esempi di progetti di altri studenti (come *HiveMind* e *TecWeb2024*), utili come riferimento per l'implementazione del proprio progetto.
- **`zip/`**: Archivi compressi contenenti risorse varie (codice d'esempio, facsimili di prove d'esame, slide).
- **`Traccia Progetti Tecnologie Web - 2024-2025.pdf`**: Documento ufficiale contenente le specifiche, i requisiti e le tracce per la realizzazione del progetto finale.

## 🚀 Progetto Principale: StreetCats

Il progetto d'esame è situato nella cartella `streetcats`. 
Per avviarlo o testarlo, si consiglia di consultare eventuali file di configurazione presenti all'interno delle rispettive cartelle `backend` e `frontend`.

## ⚙️ Tecnologie Utilizzate

*(Stack tecnologico principale del corso e del progetto)*
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla / Framework)
- **Backend**: Node.js, Express.js
- **Altro**: Strumenti di testing e moduli NPM
